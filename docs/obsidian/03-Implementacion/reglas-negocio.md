# Reglas de Negocio

- Una poliza INDIVIDUAL solo puede tener 1 riesgo.
- No se puede renovar una poliza CANCELADA.
- Cancelar poliza cancela todos sus riesgos.
- Agregar riesgo solo aplica a poliza COLECTIVA.
- Cancelar riesgo cambia estado a CANCELADO.
- Toda mutacion intenta enviar evento al CORE mock.
